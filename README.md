# React Tailwind Dashboard

A responsive dashboard built with React and Tailwind CSS.

## Features
- React (Vite)
- Tailwind CSS
- Responsive layout
- Dashboard UI (charts, tables, stat cards)
- Reusable components

## Tech Stack
- React
- Vite
- Tailwind CSS

## Installation
git remote add origin https://github.com/jaseeljazc/techon-dashboard.git
npm install

## Run Project
npm run dev

Open: http://localhost:5173

## Project Structure
src/
components/
pages/
assets/
styles/

## Customization
- Update styles in tailwind.config.js
- Edit components in src/components
- Connect APIs for dynamic data
